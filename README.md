# 언어의 혈투 (TOPIK Pro) - K-메디컬 헬스 투어 통합 플랫폼

## 🎭 프로젝트 개요

**"언어는 생존이다. 가장 치열하게 배우고, 완벽하게 지배하라."**

Netflix 킹덤 시리즈에서 영감을 받은 다크 고딕 테마의 TOPIK 학습 플랫폼과 K-메디컬 헬스 투어를 통합한 혁신적인 웹 애플리케이션입니다.

### 핵심 특징
- 🎨 **킹덤 테마 UI**: 먹물 번짐, 붉은 인장, 3D 훈민정음 애니메이션
- 🌏 **11개 언어 지원**: 한국어, 영어, 중국어, 힌디어, 스페인어, 프랑스어, 아랍어, 벵골어, 포르투갈어, 러시아어, 인도네시아어
- 🏥 **K-메디컬 헬스 투어**: 건강검진 + 한방 힐링 프로그램
- 📚 **TOPIK 완전 대비**: 진단 테스트, 학습 콘텐츠, 대학/기업 연계
- 🎓 **무료 교육**: 로그인 없이 즉시 접근 가능

## 🏥 K-메디컬 헬스 투어

### 건강검진 패키지
1. **기본 건강검진** (₩350,000)
   - 기본 신체계측, 혈액검사, 소변검사, 흉부 X-ray, 심전도 검사

2. **정밀 건강검진** (₩750,000) ⭐ 인기
   - 기본검진 + 복부 초음파, 위내시경, CT 촬영, 종양표지자, 영양 상담

3. **암 정밀검진** (₩1,200,000)
   - 정밀검진 + PET-CT, 전신 MRI, 유전자 검사, 암 전문의 상담

4. **VIP 임원 패키지** (₩2,500,000) 👑 프리미엄
   - 암검진 + 1:1 코디네이터, 프리미엄 한방, 럭셔리 호텔, 전용 차량, 미식 한정식

### 한방 힐링 프로그램
- 🌿 **소화기 케어**: 침·뜸 치료
- 💆 **근골격 케어**: 추나 요법
- 🧘 **스트레스 완화**: 약침·한방 테라피
- ☯️ **체질 분석**: 사상체질 진단

### 경산 선택의 이유
- 💰 대도시 대비 30-50% 저렴
- ⏱️ 대기 시간 최소화, 당일 검진 가능
- 🌸 한방 치료 결합
- 🏆 영남대병원, 경산중앙병원 등 신뢰 기관

## 📚 TOPIK 학습 플랫폼

### 레벨별 커리큘럼
- **Level 1 입문** (TOPIK I 1급): AI 발음 교정, 짧은 영상, 기초 문법
- **Level 2 초중급** (TOPIK I 2급): TTMIK식 문법, 실시간 회화 미션
- **Level 3 중급** (TOPIK II 3-4급): 읽기/듣기 집중, AI 작문 채점
- **Level 4 고급** (TOPIK II 5-6급): 고급 텍스트, 논술형, 원어민 면접

### 핵심 기능
- 🎯 **진단 테스트**: 20분, 30문항, 즉시 결과
- 📊 **AI 채점**: 쓰기 자동 첨삭, 점수 예측
- 🎓 **대학 연계**: 영남대 등 경산 지역 대학 협력
- 🏢 **기업 매칭**: 제조업 인력 연계

## 🛠 기술 스택

### Frontend
- **HTML5/CSS3**: 킹덤 테마 커스텀 스타일
- **JavaScript**: 바닐라 JS (CDN 기반)
- **Fonts**: Pretendard, Hahmlet, Nanum Myeongjo
- **Icons**: Font Awesome 6.4.0
- **HTTP Client**: Axios

### Backend
- **Framework**: Hono (v4.11.4)
- **Runtime**: Cloudflare Workers
- **Database**: Cloudflare D1 (SQLite)
- **Build Tool**: Vite (v6.4.1)

### Deployment
- **Platform**: Cloudflare Pages
- **Environment**: Cloudflare Workers Runtime
- **Node Compatibility**: nodejs_compat flag
- **Process Manager**: PM2 (development)
